# Genius_Micro


Este repositório é destinado ao projeto academico da disciplina EEN251-Microcontroladores e Sistemas Embarcados do Instituto Mauá de Tecnologia.

A ideia do projeto é desenvolver um jogo de tabuleiro baseado no sucesso da decada de 80 utilizando um NXP MBED LPC 1768, 4 botões, 4 LEDs RGB, 
uma bomba d'água e um modulo bluetooth e uma caixa de som (buzzer).

NXP MBED LPC 1768 é o microcontrolador principal.
Os 4 botões são responsáveis por receber as entradas de contato com o jogador.
Os 4 Leds RGB são acionados com o decorrer do jogo. 
A bomba d'água e o buzzer são os indicadores de derrota do jogador, quando o jogador erra, um som é tocado e um jato d'água
é disparado na direção do jogador.
O módulo bluetooth é responsável pelo inicio e seleção do modo de jogo (Classico e Roleta Russa).
